About

MHD-Hermes is a magnetohydrodynamics simluation software written in C++ and Python, using the Hermes hp-FEM library.

It was developed during the summer 2009 in the Los Alamos National Laboratory.

Download

This project is opensource (BSD licensed), source code is available at http://github.com/certik/mhd-hermes

You can download it in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone \
  git://github.com/certik/mhd-hermes
